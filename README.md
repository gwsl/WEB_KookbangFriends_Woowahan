# 국방프렌즈
> https://https://dev.kookbang.kr/

## 프로젝트 설명
 * '국방프렌즈'는 군생활을 하는 용사들을 대상으로 한 온라인 상담지원 서비스를 지원하는 웹사이트이다. 상담기능 외에도 군부대 내의 전 국군 장병 및 군무원을 대상으로 한 커뮤니티가 마련되어 있으며, 국방부 소식 등을 카드뉴스의 형태로 제공하여 국방부 홍보의 기능도 한다.<br>
